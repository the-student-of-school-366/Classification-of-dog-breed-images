### Классификация изображений пород собак (Computer Vision)
**Технологии**: Python, PyTorch, CNN, ResNet18, Transfer Learning, Selenium

- Собрал датасет из 1000 изображений 4 категорий через веб-скрапинг с использованием Selenium
- Реализовал CNN архитектуру с нуля для классификации изображений
- Применил Transfer Learning: fine-tuning предобученной ResNet18
- Настроил пайплайн обработки изображений (resize, нормализация, аугментации)
- Визуализировал embeddings через t-SNE для анализа представлений модели
- Достиг точности 81% на валидационной выборке с использованием ResNet18
